# [Parkway] Setup

## 1. Backend (IntelliJ IDEA)
* **Open:** Open the `backend` folder in IntelliJ.
* **Sync:** Allow Maven/Gradle to download dependencies.
* **Run:** Locate the Main Application class and click the **Run** (Play) button.
* **URL:** `http://localhost:8080` (default)

## 2. Frontend (React JS)
* **Open:** Open the `frontend` folder in VS Code.
* **Install:** Open the Terminal (`Ctrl` + `~`) and run:
    ```bash
    npm install
    ```
* **Run:** Start the development server:
    ```bash
    npm run dev
    # OR
    npm start
    ```
* **URL:** `http://localhost:3000` (default)
